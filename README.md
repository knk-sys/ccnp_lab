📘 Cisco Networking Lab

📌 Description
This lab is designed to practice essential Layer 2 and Layer 3 networking concepts using a realistic enterprise topology. 
It includes core, distribution, and access layers connected to dual ISPs for redundancy.


🎯 Topics Covered

VLAN and Inter-VLAN Routing
Trunk Configuration (802.1Q)
EtherChannel (link aggregation)
STP (MST) for loop prevention
HSRP and VRRP for gateway redundancy
Static Routing and OSPF
NAT for internet access
VRF for traffic separation


🧪 Lab Goal
To understand how different networking technologies work together to provide:

High availability
Redundancy
Efficient traffic forwarding
Scalable network design
